How to setting up Github with VScode

"VScode>File>Open Folder" open the folder containing your project

Click on the SCM icon (source control) on VScode while keep your project open

Initialize the repository (for some cases if a folder dialog box is open containing the project folder, then click ok)

You will have a .git folder in your project folder, if you cannot see that in VScode check my vscode folder here

To see .git>config file in VScode, change the File>preference>settings> set git to false either in user or workspace

Also, if you want to open the html file from iside the browser then ctrl+shift+p, write: Configure Tasks> click tasks.json> other tasks

Replace the existing code with my code in tasks.json

Create github repository without initialize and then copy the link of your project. Type the below in the VS terminal::

git remote add origin https://github.com/tanvir03/github1.git

git push -u origin master

Make any change in VScode, you can commit by writing a message and click on the checkbox (SCM). For sync, click ... and sync

You can also push from your VScode project. SCM>...>Push
