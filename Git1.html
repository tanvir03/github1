How to setting up Github with VScode

Click on the SCM icon (source control) on VScode while keep your project open

Initialize

You will have a .git folder in your project folder, if you cannot see that in VScode check my vscode here

To see .git>config file in VScode, change the File>preference>settings> git to false

Also, if you want to open the html file from iside the browser then ctrl+shift+p, Configure Tasks, other tasks, use my code in tasks.json

create github repository without initialize and then copy the link. Type the below in the VS terminal::

git remote add origin https://github.com/tanvir03/github1.git

git push -u origin master

Make any change in VScode, you can commit by writing a message and click on the checkbox (SCM). For sync, click ... and sync
